    Unfollowing Non-Followers on Instagram with Instagrapi¶

    Introduction¶

    Using Instagrapi (an unofficial Instagram API wrapper for Python), you can unfollow everyone who doesn't follow you back on Instagram. Here's a step-by-step guide on how to do it:

    Please note that using unofficial Instagram API wrappers is against Instagram's terms of service. This means that your account could be banned if you use this script. Use at your own risk.

    Installation¶

    First, you need to install the Instagrapi library. You can do this using pip:

    pip install instagrapi

    Creating an Instagrapi Client¶

    Next, you need to create an Instagrapi client. You can do this by importing the Client class from the instagrapi library and creating an instance of this class:

    from instagrapi import Client

username = 'your_username'
password = 'your_password'

client = Client()
client.login(username, password)

    Getting a List of Your Followers and Following¶

    To determine who doesn't follow you back on Instagram, you need to get a list of your followers and following. You can do this by calling the get_followers and get_following methods on your Instagrapi client:

    followers = client.user_followers(username)
following = client.user_following(username)

    Unfollowing Non-Followers¶

    After getting a list of your followers and following, you need to determine who doesn't follow you back on Instagram. You can do this by looping through your following and checking if each user is in your followers. If they are not in your followers, then you need to unfollow them. You can do this by calling the unfollow method on your Instagrapi client:

    for user in following:
    if user not in followers:
        client.unfollow(user)

    Conclusion¶

    Using Instagrapi, you can unfollow everyone who doesn't follow you back on Instagram. This can be useful if you want to clean up your Instagram account and remove people who don't follow you back.
